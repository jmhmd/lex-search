Lex Search

Fast search of standardized lexicons like ICD10 and Radlex, using flat files, node.js + express.js, Twitter typeahead.js and lunr.js

Inspired by ClinDesk icd10 search, but doing the searching server side, in-memory, to avoid needing to download a multi-MB file to the client. This also theoretically allows many more lexicons to be added without affecting performance on the client.

Source and docs: http://github.com/jmhmd/lex-search
